El Viaje de una Solicitud: De la URL a la Pantalla

1. Las Direcciones IP: El documento de identidad

Cada dispositivo conectado a Internet tiene un identificador único llamado Dirección IP (ej. 192.0.2.1). Los servidores también tienen una, y es la única forma en que las computadoras saben exactamente con quién están hablando.

2. El Sistema de Nombres de Dominio (DNS)

Como recordar números complejos como 142.250.190.46 es difícil para los humanos, inventamos los nombres de dominio (como google.com).

¿Qué es el DNS? Funciona como la libreta de contactos de tu teléfono. Cuando escribís una URL, tu navegador le pregunta al servidor DNS: "¿Cuál es la dirección IP de este dominio?". El DNS le responde con la IP exacta para que el navegador pueda conectarse al servidor adecuado.
